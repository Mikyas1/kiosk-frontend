# Kiosk frontend

## Configuration

add a `.env.local` file on the root of the folder with the content of

```
VUE_APP_API_URL="http://localhost:8000"
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
